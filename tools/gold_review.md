# 词对齐人工标注审阅包(100 句)

## 标注方法(只动错的行)
- 每行格式:`- [ ] 英文词 = 对齐结果`(`(灰)`表示系统判定不上色)
- **对的行:不用动**
- **错的行:把 [ ] 改成 [x],行尾加 → 正确答案**(应该灰色就写 → 灰)
- 例:`- [x] keep = 让 → 保持`
